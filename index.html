Key Contributions

Local Scale Equivariance

We model realistic, spatially-varying scale changes rather than uniform global scaling.

Monotone Scaling Group

We approximate local scaling using invertible monotonic functions that form a transformation group.

Deep Equilibrium Canonicalizer (DEC)

We introduce a novel canonicalizer using deep equilibrium models instead of explicit optimization.

Plug-and-Play Boost

Our method improves scale robustness and accuracy across models with minimal overhead.

Monotone Scaling: A Clean Approximation

1D Intuition: Stretching a Rubber Band

Consider a function \( f(x) \) on \([0,1]\). Global scaling rescales the domain uniformly:

\( R_a[f](x) = f(a^{-1} x) \)

To capture non-uniform stretch, we introduce a strictly increasing warp \( l \) and define:

\( S(f; l)(x) = f(l^{-1}(x)) \)

Intuition ‚Äî imagine a rubber band: instead of pulling both ends evenly, you stretch some segments more than others, but never fold it. The derivative \( dl/dx \) reveals the local stretch / compression rate.

2D Extension: Warping an Image Grid

Extending to images \( I(x,y) \), we apply an invertible warp \( l(x,y)=(l_X, l_Y) \):

\( S(I; l)(x,y) = I(l^{-1}(x,y)) \)

Horizontal component \( l_X \) controls left‚Äìright stretch, while \( l_Y \) controls vertical. The warp bends the grid smoothly‚Äîlines never intersect.

Lemma 2. The family of such monotone, Jacobian-commutative warps forms a group: closed, invertible, and containing the identity. This property enables principled equivariant modeling.

Deep Equilibrium Canonicalizer (DEC)

DEC amortizes the expensive optimization usually required for canonicalization. A DEQ module predicts optimal warp parameters as a fixed point of a learned update rule, applied inside the network's feature space.

  1. 1.

    Feature map \(F_k\) enters the DEC.

  2. 2.

    The DEQ solver rapidly predicts \(\Phi_k\), the local scaling parameters.

  3. 3.

    We canonicalize the features via \(S^{-1}(F_k; \Phi_k)\).

  4. 4.

    The standard layer processes the normalized features.

  5. 5.

    The warp is re-applied so the next layer receives equivariant representations.
